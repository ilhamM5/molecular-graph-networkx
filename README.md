### Running the Notebook

To get started:

1. **Run all cells** in the notebook.
2. Navigate to the **last cell** to launch and interact with the graphical user interface (GUI).
3. All non-essential cells are initially hidden to make navigation to the GUI easier provide a smoother experience.
